EcoSentinel Dashboard

To run the EcoSentinel dashboard locally, follow these steps:

Option 1: Using Python (Recommended)

Open your terminal in the webapp folder and run:

python -m http.server 8000

Then visit: http://localhost:8000

Option 2: Using Node.js

Install and run a simple HTTP server:

npm install -g http-server
http-server . -p 8000

Option 3: Using VS Code Live Server

Install the "Live Server" extension in VS Code and right-click on index.html ‚Üí "Open with Live Server"
